============
Installation
============


Stable release
--------------

To install apkutils, run this command in your terminal:

.. code-block:: console

    $ pip install apkutils

This is the preferred method to install apkutils, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for apkutils can be downloaded from the `Gitee repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://gitee.com/kin9-0rz/apkutils

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://gitee.com/kin9-0rz/apkutils/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Gitee repo: https://gitee.com/kin9-0rz/apkutils
.. _tarball: https://gitee.com/kin9-0rz/apkutils/tarball/master
